prism is a waveform player and effects processor.

it slices audio into rhythmic 4-bar loops.  

**controls**

left mouse to seek across waveform.  

**buttons**
- play/pause 
- clear
- process  
- export  

**sliders**    
- bpm
- grid density
- swing
- vol/pan  
- playback rate  
- tone 
- crush
- sample rate
- filter
- reverb 

this program is free and [donations are welcome](https://ko-fi.com/geltz)
