prism is a waveform player and effects processor with a qt6 interface.

it takes audio samples, slices them into pieces, and rearranges them into rhythmic 2-bar loops.  

**controls**

left mouse to seek across waveform.  

**buttons**
- play/pause/stop  
- clear
- process  
- export  

**sliders**    
- grid density
- bpm
- spectral wash  
- bit crush  
- playback rate  
- sample rate   
- glue reverb  
- filter mod  
- vol/pan mod
